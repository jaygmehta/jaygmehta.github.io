    The set of zeros of a non-constant entire function is    .
    i)
    uncountable
    ii)
    countable
    iii)
    finite
    iv)
    countably infinite

    Choice i) is Incorrect.
    Choice ii) is Correct!
    Choice iii) is Incorrect.
    Choice iv) is Incorrect.
    Logical argument looking at the options. Consider the entire function f(z) = z. It has only one zero (finite or countable) at z = 0. So the other two options, that is, uncountable and countably infinite are not correct. Consider the entire function f(z) = sin ⁡ z. It has countably infinite zeros at z = nπ, n ∈ ℤ. So the finite option is not correct. So the only choice left is “countable” (which works for countably infinite and also for finite set of zeros). Let f : ℂ → ℂ be a non-constant entire function. For each n ∈ ℕ, let Bn = B(0,n) be the open ball with center 0 and radius n. Then note that ℂ = ⋃ n=1∞B n. If for some k, Bk contains infinitely many zeros of f, then by the Bolzano-Weierstrass property, it has a limit point in Bk. Then by the identity theorem, f ≡ 0 which is contradiction to the given fact that f is non-constant. Hence, the set of zeros of f is countable. Functions like f(z) = z and f(z) = ez need not have countably infinite zeros and hence that option is not valid.

    If f is a nonconstant entire function, then the set of zeros of f is
    i)
    countably infinite
    ii)
    countable
    iii)
    finite
    iv)
    uncountable

    Choice i) is Incorrect.
    Choice ii) is Correct!
    Choice iii) is Incorrect.
    Choice iv) is Incorrect.
    For each n ∈ ℕ, let Bn = B(0,n). Then ℂ = ⋃ n=1∞B n. For each n, Bn contains at most finitely many zeros of f. If for some k, the bounded set Bk contains infinitely many zeros of f, then by Bolzano-Weierstrass property, the set of zeros of f in Bk has a limit point in Bk. By identity theorem, f ≡ 0 which is a contradiction to the fact that f is a nonconstant function.
    Thus, each Bn contains at most finitely many zeros of f. Hence, the set of zeros of f in ℂ is a countable set. However, the function f(z) = z has only one zero at z = 0. Therefore, the set of zeros of f need not be necessarily countably infinite. On the other hand, f(z) = sin ⁡ z has zeros at nπ for n ∈ ℤ which means that the set of zeros of f need not be necessarily a finite set also.
    Hence, it would be appropriate to say that the set of zeros of f is countable.
